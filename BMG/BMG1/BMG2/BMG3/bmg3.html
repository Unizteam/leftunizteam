"Left-Sight" Başlığını ve Metin Formatlama

1. HTML'de Başlık Nasıl Ayarlanır?

Web sitenin başlığını "Left-Sight" olarak belirlemek için aşağıdaki HTML kodunu kullanabilirsin:

HTML Başlık Ayarı

                    
                        <title>Left-Sight</title>

2. HTML İçinde Belirli Kelimeleri Nasıl Biçimlendirebiliriz?

Aşağıdaki örnek, HTML'de belirli kelimeleri nasıl bold, kırmızı ve italik yapabileceğini gösterir.

HTML Metin Biçimlendirme

                    
                        <p>
                            Html içinde bir cümle kurmak ve bazı kelimeleri 
                            <b>bold</b> ve 
                            <span style="color: red;">kırmızı</span> ve 
                            <i>italik</i> yapmak.
                        </p>

3. Alternative Method: CSS ile Stil Tanımlamak

Eğer bu kelimeleri CSS sınıfları kullanarak biçimlendirmek istiyorsanız, aşağıdakileri kullanabilirsiniz:

CSS ile Metin Biçimlendirme

                    
                        .bold{
                            font-weight: bold;
                        }

                        .red {
                            color: red;
                        }

                        .italic {
                            font-style: italic;
                        }

Sorunun standart cevabı: 


            <!DOCTYPE html>
            <html lang="tr">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Left-Sight</title>
            </head>
            <body>
                <p>Html içinde bir cümle kurmak ve bazı kelimeleri <b>bold</b> ve <span class="red">kırmızı</span> ve <i>italik</i> yapmak. </p>
            </body>
            </html>
